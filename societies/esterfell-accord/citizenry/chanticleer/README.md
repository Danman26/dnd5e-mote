# Chanticleer

**Chanticleer** runs the [Bridgeport](../../bridgeport) chapter of [Reynard's Den](../../../../organizations/reynards-den).

## Attributes

- pronouns: he/him
- species: aarakocra
- languages: Common, Auran, Elvish
